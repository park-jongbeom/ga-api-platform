🎯 MVP 핵심 기능 범위

포함된 핵심 기능 (7개)

  • 회원가입/로그인 - 사용자 인증
  • 필수 정보 입력 - 학력, 개인정보, 유학 목표 (3단계 입력)
  • AI 매칭 실행 - 6대 지표 기반 매칭 알고리즘
  • 추천 학교 Top 5 표시 - 매칭률 순위별 학교 리스트
  • 학교 상세 보기 - 학교 정보 + 설명 가능한 추천 이유
  • 지원 관리 - 보관하기/지원하기 기능
  • 대시보드 - 매칭 현황 및 지원 현황 요약

제외된 기능 (Phase 2 이후로 연기)

  • 4개 프로그램 별 상세 페이지
  • 문서 표준화 (지원/비자 서류)
  • 미국 생활 정보 허브
  • 고급 검색 기능
  • PDF 생성/공유 기능
  • 챗봇 문의
제외 이유: 사용자가 AI 매칭을 체험하고 추천 결과를 확인할 수 있는 최소 기능에 집중하여, 빠른 시장 검증과 피드백 수집을 우선합니다.

백엔드 개발 일정 (6주, 57시간)

주차 주요 작업 시간 프론트엔드 블로킹 해소
Week 1 Mock API + Swagger 문서 10시간 ✓ 프론트 작업 즉시 시작 가능
Week 2 사용자 프로필 API + DB 마이그레이션 10시간 ✓ 실제 프로필 저장 가능
Week 3 AI 매칭 엔진 핵심 로직 (6대 지표) 9시간 -
Week 4 매칭 실행 API + 설명 생성 10시간 ✓ 실제 매칭 결과 연동
Week 5 지원 관리 + 대시보드 API 10시간 ✓ 지원 기능 연동
Week 6 보안 강화 + 모니터링 8시간 -

프론트엔드 개발 일정 (2.5주, 60시간)

주차 주요 작업 시간 백엔드 의존성
Week 1 회원가입 + 프로필 입력 UI 24시간 Mock API (Week 1)
Week 2 6대 지표 차트 + 학교 리스트 24시간 실제 API (Week 4)
Week 3 지원 관리 + 대시보드 + 테스트 12시간 Application API (Week 5)

병렬 개발 전략

백엔드 Week 1 → Mock API 제공 ↓ 프론트 Week 1 → Mock 데이터로 UI 개발 시작 (블로킹 해소!) ↓ 백엔드 Week 2~4 → 실제 API 개발 진행 ↓ 프론트 Week 2~3 → 실제 API 연동 및 완성

핵심 포인트

  • 백엔드가 Week 1에 Mock API를 제공하여 프론트엔드가 즉시 작업 시작
  • 프론트엔드는 2.5주 만에 완료 (백엔드 Week 4 시점)
  • 백엔드 Week 4~6에 실제 API를 완성하면서 프론트엔드 통합
